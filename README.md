# TP Formulaire et Tableau en PHP

## Description
Ce projet est un exercice pratique en PHP permettant :
- de saisir les informations d’une personne via un formulaire HTML,
- d’enregistrer ces données dans un fichier JSON,
- d’afficher la liste des personnes dans un tableau HTML.

## Fonctionnalités
- Formulaire d’ajout de personne (prénom, nom, adresse, téléphone)
- Stockage des données dans un fichier `personne.json`
- Affichage dynamique des personnes enregistrées
- Conservation des données après rechargement de la page

## Technologies utilisées
- PHP
- HTML5
- Bootstrap
- JSON
